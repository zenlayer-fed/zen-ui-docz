配置

Icon支持按需加载，请配置babel配置文件，添加@zenlayer/babel-plugin-importSvg。

.babelrc

{
  ...
  "plugins": [
    [
      "@zenlayer/babel-plugin-importSvg",
      {
        "displayName": "Icon",
        "propName": "type",
        "directory": "/assets/",
        "libraryName": "@zenlayer/zen-ui"
      }
    ],
    ...
  ]
}

Icon依赖svg-sprite-loader，请在项目中配置，包括但不限于webpack、gatsby等构建工具。

webpack.config.js

module.exports = {
  module: {
      rules: [
        {
          test: /\.svg$/,
          loader: 'svg-sprite-loader',
          include: /\@zenlayer\/zen-ui/,//如项目中svg使用其他形式，请保证zen-ui下使用svg-sprite-loader。
        },
        ...
      ]
  }
  ...
}

gatsby-config.js

module.exports = {
  plugins: [
    'gatsby-plugin-svg-sprite',
    ...
  ],
  ...
};

API

Icon

属性说明类型默认值
styleStyle properties of icon, like fontSize and colorCSSProperties-
classNameclassName of Iconstring-
spinRotate icon with animationbooleanfalse
typeThe zen-ui/assets file name of SVGstring-

CustomIcon

属性说明类型默认值
styleStyle properties of icon, like fontSize and colorCSSProperties-
spinRotate icon with animationbooleanfalse
rotateRotate degrees (not working in IE9)number-
componentThe component used for the root node.React.Element-

The following properties are available for the component:

属性说明类型默认值
styleThe computed style of the svg elementCSSProperties-
classNameThe computed class name of the svg elementstring-
fillDefine the color used to paint the svg elementstring'currentColor'
heightThe height of the svg elementstring | number'1em'
widthThe width of the svg elementstring | number'1em'

注意

因为icon的按需加载是在编译时完成，所以请注意一些运行时的用法

//无效
import iconsMap from './iconsMap.ts'
const IconCom = () => { 
    return (
      <>
      {
        iconsMap.map(type=><Icon type={type} />)
      }
      </>
    )
  }}
//无效
const iconsmap = {
  xx1:Close,
  xx2:StepBackwardOutlined,
  xx3:UpCircleFilled
}
...
<Icon type={iconsmap[type]} />
...
//正确写法
const iconsmap = {
  xx1:<Icon type="Close" />,
  xx2:<Icon type="StepBackwardOutlined" />,
  xx3:<Icon type="UpCircleFilled" />
}
...
{iconsmap[type]}
