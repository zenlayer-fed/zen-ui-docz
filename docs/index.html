zen-ui
zen-ui of react安装配置
组件

安装

$ npm i @zenlayer/zen-ui -D

配置

支持cjs/esm形式，生成ES module能够更好地进行tree shaking, 配置babel-plugin-import支持js和css按需加载。

.babelrc

{
  "plugins": [
    [
      "babel-plugin-import",
      {
        "libraryName": "@zenlayer/zen-ui",
        "libraryDirectory": "esm",
        "style": true
      }
    ],
    ...
  ]
  ...
}

Icon依赖svg-sprite-loader，即使不使用Icon，组件中也会依赖Icon，所以请在项目中配置，包括但不限于webpack、gatsby等构建工具。

webpack.config.js

module.exports = {
  module: {
      rules: [
        {
          test: /\.svg$/,
          loader: 'svg-sprite-loader',
          include: /\@zenlayer\/zen-ui/,//如项目中svg使用其他形式，请保证zen-ui下使用svg-sprite-loader。
        },
        ...
      ]
  }
  ...
}

gatsby-config.js

module.exports = {
  plugins: [
    'gatsby-plugin-svg-sprite',
    ...
  ],
  ...
};
